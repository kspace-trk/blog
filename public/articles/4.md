## github actionsを設定する
github側に環境変数のsecretsを設定した
rsyncでビルド済みのものをデプロイしたい
burnett01/rsync-deploymentsを使用する。
https://github.com/Burnett01/rsync-deployments

`burnett01/rsync-deployments@5.2` で設定を試みたが、`rsync: [sender] change_dir "/github/workspace/hoge" failed: No such file or directory (2)`というエラーがでてうまくいかなかった。

`burnett01/rsync-deployments@5.3` というノリでv5.3を指定することでうまく通るようになった。

書いたyamlファイルは以下の通り
```
name: deploy
on:
  push:
    branches: [ master ]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '16'
      - run: npm install
      - run: npm run build
      - name: rsync deployments
        uses: burnett01/rsync-deployments@5.3
        with:
          switches: -avzr --delete
          path: out/
          remote_path: ${{ secrets.DEPLOY_DIR }}
          remote_host: ${{ secrets.DEPLOY_HOST }}
          remote_user: ${{ secrets.DEPLOY_USER }}
          remote_key: ${{ secrets.DEPLOY_KEY }}
```
